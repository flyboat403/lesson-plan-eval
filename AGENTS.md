## 📦 项目概述
本目录包含 **教案评估 (Vocational Lesson Audit)** Skill 的核心配置文件。
该 Skill 专为**职业学校 (中职/高职)** 教师设计，旨在利用 AI 扮演"逆向教研员"，对教案进行严谨的挑刺、评估与重构，确保教学设计符合"岗课赛证融通"和"行动导向"的职教要求。

---

## 🚀 触发与使用
*   **触发词**: `教案评估`、`教案检查`、`教学设计改进`、`教案评分`。
*   **输入方式**: 支持直接粘贴教案文本，或上传 `.docx` / `.pdf` / `.md` 文件。
*   **执行逻辑**: 
    1.  **解析与守门 (Dispatcher)**: 检查教案完整性，若缺失核心要素则提供补全指南，而非强行评估。
    2.  **专家矩阵审计**: 并行执行目标、问题链、活动、资源、时间 5 维深度审查，结合职教标准（布鲁姆分类学、5E 模式、行动导向六步法）。
    3.  **输出报告**: 生成 Markdown 格式的逆向审计报告（包含评级、亮点、硬伤表、重构问题链及改进路线图）。

---

## 🛠️ 对 AI Agent 的指示
当用户触发上述关键词时，必须完整加载并遵循 `SKILL.md` 中的指令集进行响应。
*   **原则**: 坚决避免"讨好型"回复，执行"逆向挑刺"。
*   **标准**: 评估必须体现职教特色，重点关注"理实比例"、"岗位对接"和"资源可行性"。
*   **格式**: 严格遵循 `SKILL.md` 规定的 Markdown 输出模板，保持结构化排版。
