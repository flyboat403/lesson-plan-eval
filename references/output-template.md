# 输出模板字段规范

> 本文档定义教案评估报告的详细输出格式和字段规范。

---

## 报告整体结构

```
1. 评级概览
2. 分项得分表
3. 课堂结构简表
4. 核心硬伤表
5. 教法匹配诊断
6. 问题链优化表
7. 亮点保留
8. 改进路线图
```

---

## 表格1：分项得分表

| 字段 | 必填 | 格式要求 | 示例值 |
|------|------|---------|-------|
| **维度** | ✓ | 标准名称（带字母前缀） | "A 基本信息审计" |
| **得分** | ✓ | 整数，0-满分 | "8" |
| **满分** | ✓ | 整数 | "10" |
| **主要问题** | ✓ | 简要描述 | "课堂类型未明确标注、重难点缺少突破措施" |

**列宽建议**: 维度25% / 得分15% / 满分15% / 主要问题45%

**八维度满分**:
- A 基本信息审计: 10分
- B 学情审计: 12分
- C 教学目标审计: 15分
- D 教法审计: 10分
- E 活动审计: 15分
- F 问题链审计: 12分
- G 辅助审计: 12分
- H 教学评一致性审计: 14分

---

## 表格2：核心硬伤表

| 字段 | 必填 | 格式要求 | 示例值 |
|------|------|---------|-------|
| **序号** | ✓ | 整数，从1开始 | "1" |
| **位置** | ✓ | 具体到段落/环节 | "教学目标第2点" |
| **类型** | ✓ | 标准分类词 | "目标模糊/时间失衡/伪互动/形式活动/教法不匹配/学习结果误判" |
| **扣分** | ✓ | 整数，带负号 | "-3" |
| **诊断分析** | ✓ | 解释为何有问题 | "\"熟悉...\"无法考核，脱离职教要求" |
| **改进建议** | ✓ | 具体可操作 | "改为\"能独立使用万用表排查故障\"" |

**列宽建议**: 序号8% / 位置15% / 类型12% / 扣分10% / 诊断27% / 建议28%

---

## 表格3：问题链优化表

| 字段 | 必填 | 格式要求 | 示例值 |
|------|------|---------|-------|
| **原问题** | ✓ | 教案中的原始提问 | "这个零件是什么材质的？" |
| **认知层级** | ✓ | 布鲁姆层级 | "记忆/理解/应用/分析/评价/创造" |
| **优化问题** | ✓ | 改进后的提问 | "通过观察断口形貌，推测它可能是什么材质？" |
| **追问** | ○ | 可追加的引导问题 | "如果是，为什么会有这种光泽？" |
| **能力** | ✓ | 训练的核心能力 | "观察与推断能力" |

**列宽建议**: 原问题18% / 认知层级14% / 优化问题20% / 追问16% / 能力16%

---

## 表格4：课堂结构简表

| 字段 | 必填 | 格式要求 | 示例值 |
|------|------|---------|-------|
| **环节** | ✓ | 教学阶段名称 | "导入/探究/讲授/练习/评价/总结" |
| **教师活动** | ✓ | 教师主导行为 | "播放案例视频，引出本课核心问题" |
| **学生活动** | ✓ | 学生主体行为 | "观看视频，思考并分享关键词" |
| **设计意图** | ✓ | 该环节目的 | "创设情境，激发兴趣，激活自我系统" |
| **时间** | ✓ | 分钟数 | "5 min" |
| **工作属性** | ○ | 【仅专业课】资讯/计划/决策/实施/检查/评估 | "资讯" |

**列宽建议**: 环节12% / 教师25% / 学生25% / 意图20% / 时间10% / 工作属性(专业课加)8%

---

## 表格5：教法匹配诊断

| 字段 | 必填 | 格式要求 | 示例值 |
|------|------|---------|-------|
| **学习结果类型** | ✓ | 加涅五类：言语信息/智慧技能/认知策略/动作技能/态度 | "智慧技能" |
| **识别依据** | ✓ | 判断依据说明 | "目标包含\"应用规则排查故障\"，属规则应用层次" |
| **教法匹配状态** | ✓ | 匹配/偏差/缺失 | "偏差" |
| **偏差详情** | ○ | 具体偏差描述（状态为偏差时必填） | "纯讲授无练习，学生无动手应用机会" |
| **改进建议** | ✓ | 具体可操作的教法调整建议 | "增加案例分析+分组实操环节，设置故障排查任务让学生动手练习" |

**列宽建议**: 学习结果类型18% / 识别依据22% / 匹配状态12% / 偏差详情24% / 改进建议24%

**教法匹配检查对照表**:

| 学习结果类型 | 适宜教法 | 禁忌教法 |
|-------------|---------|---------|
| 言语信息 | 讲授、阅读、先行组织者 | 缺乏意义建构的机械背诵 |
| 智慧技能 | 概念辨析、规则讲解+练习、问题解决示范 | 纯讲授无练习应用 |
| 认知策略 | 策略示范、元认知引导、反思训练 | 直接灌输策略名称 |
| 动作技能 | 示范—练习—反馈闭环、分步训练 | 纯讲解无实操 |
| 态度 | 榜样示范、体验活动、正向强化、选择情境 | 说教式灌输 |

## 评级概览格式

```markdown
> **⭐综合评级**: ⭐⭐⭐☆☆ (优秀/良好/合格/待改进)
> **📍总分**: XX/100分
> **🎯课程类型**: 专业课/公共基础课（已识别）
> **🔍诊断模式**: 全面逆向审计（八维度）
```

---

## 亮点保留格式

```markdown
> **✓ 亮点1**: [具体亮点描述]
> - **所在位置**: [环节/段落]
> - **亮点价值**: [为何值得保留]
>
> **✓ 亮点2**: [具体亮点描述]
> - **所在位置**: [环节/段落]
> - **亮点价值**: [为何值得保留]
>
> **✓ 亮点3**: [具体亮点描述]
> - **所在位置**: [环节/段落]
> - **亮点价值**: [为何值得保留]
```

---

## 改进路线图格式

```markdown
> **微观修正**（本课可直接改）: 
> - [具体可立即执行的修改]
> - [另一条即时修正]
>
> **中观重构**（本单元内调整）:
> - [涉及单元层面的问题]
> - [需要重构的流程]
>
> **宏观研讨**（建议教研组讨论）:
> - [需团队协作解决的问题]
> - [制度/标准层面的议题]
```

## 字段填写禁忌

| 字段类型 | 禁止行为 | 正确做法 |
|---------|---------|---------|
| 位置 | ❌ "目标部分" | ✓ "教学目标第2点" |
| 类型 | ❌ "有问题" | ✓ "目标模糊/教法不匹配/学习结果误判" |
| 扣分 | ❌ "扣了3分" | ✓ "-3" |
| 建议 | ❌ "改一下" | ✓ "改为'能独立操作XX'" |
| 时间 | ❌ "一会儿" | ✓ "5 min" |
| 学习结果类型 | ❌ 空值或未识别 | ✓ 明确识别为加涅五类之一 |
| 教法匹配状态 | ❌ "差不多" | ✓ 明确填写"匹配/偏差/缺失" |
| 偏差详情 | ❌ 状态为偏差时留空 | ✓ 具体说明偏差表现 |
