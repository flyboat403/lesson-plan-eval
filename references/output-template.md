# 输出模板字段规范

> 本文档定义教案评估报告的详细输出格式和字段规范。

---

## 报告整体结构

```
1. 评级概览
2. 分项得分表
3. 亮点保留
4. 核心硬伤表
5. 问题链优化表
6. 课堂结构简表
7. 改进路线图
```

---

## 表格1：分项得分表

| 字段 | 必填 | 格式要求 | 示例值 |
|------|------|---------|-------|
| **维度** | ✓ | 标准名称 | "目标审计" |
| **得分** | ✓ | 整数，0-满分 | "17" |
| **满分** | ✓ | 整数 | "20" |
| **主要问题** | ✓ | 简要描述 | "核心问题偏离主题、目标表述不够精确" |

**列宽建议**: 维度25% / 得分15% / 满分15% / 主要问题45%

---

## 表格2：核心硬伤表

| 字段 | 必填 | 格式要求 | 示例值 |
|------|------|---------|-------|
| **序号** | ✓ | 整数，从1开始 | "1" |
| **位置** | ✓ | 具体到段落/环节 | "教学目标第2点" |
| **类型** | ✓ | 标准分类词 | "目标模糊/时间失衡/伪互动/形式活动" |
| **扣分** | ✓ | 整数，带负号 | "-3" |
| **诊断分析** | ✓ | 解释为何有问题 | "\"熟悉...\"无法考核，脱离职教要求" |
| **改进建议** | ✓ | 具体可操作 | "改为\"能独立使用万用表排查故障\"" |

**列宽建议**: 序号8% / 位置15% / 类型12% / 扣分10% / 诊断27% / 建议28%

---

## 表格3：问题链优化表

| 字段 | 必填 | 格式要求 | 示例值 |
|------|------|---------|-------|
| **原问题** | ✓ | 教案中的原始提问 | "这个零件是什么材质的？" |
| **认知层级** | ✓ | 布鲁姆层级 | "记忆/理解/应用/分析/评价/创造" |
| **优化问题** | ✓ | 改进后的提问 | "通过观察断口形貌，推测它可能是什么材质？" |
| **追问** | ○ | 可追加的引导问题 | "如果是，为什么会有这种光泽？" |
| **能力** | ✓ | 训练的核心能力 | "观察与推断能力" |

**列宽建议**: 原问题18% / 认知层级14% / 优化问题20% / 追问16% / 能力16%

---

## 表格4：课堂结构简表

| 字段 | 必填 | 格式要求 | 示例值 |
|------|------|---------|-------|
| **环节** | ✓ | 教学阶段名称 | "导入/探究/讲授/练习/评价/总结" |
| **教师活动** | ✓ | 教师主导行为 | "播放《哪吒》片段，引出主题" |
| **学生活动** | ✓ | 学生主体行为 | "观看视频，思考分享关键词" |
| **设计意图** | ✓ | 该环节目的 | "创设情境，激发兴趣" |
| **时间** | ✓ | 分钟数 | "5 min" |
| **工作属性** | ○ | 【仅专业课】资讯/计划/决策/实施/检查/评估 | "资讯" |

**列宽建议**: 环节12% / 教师25% / 学生25% / 意图20% / 时间10% / 工作属性(专业课加)8%

---

## 评级概览格式

```markdown
> **⭐综合评级**: ⭐⭐⭐☆☆ (优秀/良好/合格/待改进)
> **📍总分**: XX/100分
> **🎯课程类型**: 专业课/公共基础课（已识别）
> **🔍诊断模式**: 全面逆向审计
```

---

## 改进路线图格式

```markdown
> **微观修正**（本课可直接改）: 
> - [具体可立即执行的修改]
> - [另一条即时修正]

> **中观重构**（本单元内调整）:
> - [涉及单元层面的问题]
> - [需要重构的流程]

> **宏观研讨**（建议教研组讨论）:
> - [需团队协作解决的问题]
> - [制度/标准层面的议题]
```

---

## 字段填写禁忌

| 字段类型 | 禁止行为 | 正确做法 |
|---------|---------|---------|
| 位置 | ❌ "目标部分" | ✓ "教学目标第2点" |
| 类型 | ❌ "有问题" | ✓ "目标模糊" |
| 扣分 | ❌ "扣了3分" | ✓ "-3" |
| 建议 | ❌ "改一下" | ✓ "改为'能独立操作XX'" |
| 时间 | ❌ "一会儿" | ✓ "5 min" |
