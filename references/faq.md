# 常见问题 FAQ

> 本文档解答教案评估过程中常见疑问。

---

## 课程类型识别

**Q1: 如何区分专业课和公共基础课？**

A: 按以下关键词判定：
- **专业课关键词**: 实训、实操、技能、岗位、工艺、操作、故障、维修、加工、检测、护理、烹饪、汽修等
- **公共基础课关键词**: 心理健康、思政、语文、数学、英语、体育、艺术、劳动、职业规划、就业指导等

**Q2: 课程名称不明确怎么办？**

A: 查看教案内容：
- 有岗位对接描述 → 专业课
- 有实训/实操环节 → 专业课
- 无岗位描述、偏理论 → 公共基础课

## 评估流程问题

**Q3: 用户只给教案片段怎么处理？**

A: 执行《教案补全指南》而非强行评估：
- 输出"请补充：①目标②重难点③过程"
- 补充后自动触发重新评估

**Q4: 一份教案有多个版本，评估哪个？**

A: 评估用户提供的那一版。若有版本说明，可询问"需要评估哪个版本？"

**Q5: 教案是英文/双语怎么办？**

A: 正常评估，输出报告用中文，但保留英文专业术语。

## 扣分计算问题

**Q6: 同一个问题重复扣分吗？**

A: 不重复。例如：
- "目标散无核心问题"已扣-5
- 不额外扣该目标的"模糊词"

**Q7: 八维度扣分上限如何规定？**

A: 各维度扣分上限如下：

| 维度 | 维度名称 | 满分 | 扣分上限 |
|------|---------|------|---------|
| A | 基本信息审计 | 10分 | -10分 |
| B | 学情审计 | 12分 | -12分 |
| C | 教学目标审计 | 15分 | -15分 |
| D | 教法审计 | 10分 | -10分 |
| E | 活动审计 | 15分 | -15分 |
| F | 问题链审计 | 12分 | -12分 |
| G | 辅助审计 | 12分 | -12分 |
| H | 教学评一致性审计 | 14分 | -14分 |

**Q8: 多个轻微问题累加超过单项上限？**

A: 按上限截断：
- 错别字/语病：每处0.5分，上限-1分
- 伪互动提问：每处1分，上限-2分
- 模糊目标词：每词1分，上限-3分

## 八维度专项问题

### 课堂结构与输出格式

**Q9: 课堂结构表必须含"工作属性"列吗？**

A: 仅专业课需要。公共基础课省略此列。

**Q10: 硬伤表最多列几条？**

A: 建议3-7条，聚焦核心问题，不列琐碎小问题。

**Q11: 问题链优化表何时必须输出？**

A: 发现以下问题时必输出：
- 存在伪互动提问
- 问题缺少思维递进
- 无追问机制
- 问题链与教学目标不匹配

---

### 教法审计专项

**Q12: 如何识别加涅五类学习结果类型？**

A: 按教学活动特征判定：
- **言语信息**：陈述性知识传授，如"介绍概念"、"讲解定义"、"列举特点"
- **智慧技能**：规则/程序应用，如"运用公式"、"解题步骤"、"故障诊断"
- **认知策略**：学习策略培养，如"记忆方法"、"思维技巧"、"自我监控"
- **动作技能**：操作性技能形成，如"实操练习"、"操作演示"、"技能训练"
- **态度**：情感态度改变，如"职业认同"、"安全意识"、"团队合作"

**Q13: 教法偏差如何判定？**

A: 以下情况判定为教法偏差：
- 智慧技能教学无练习环节 → 扣维度D 3-5分
- 动作技能教学纯讲解无示范 → 扣维度D 5分（触发NEVER规则）
- 认知策略教学仅告知结论不引导发现 → 扣维度D 3分
- 言语信息教学超时（>40%）仍无互动 → 扣维度D 2分

**Q14: 态度类教学如何评估？**

A: 态度评估标准：
- 正面案例：情境体验、角色扮演、案例讨论、职业感悟
- 负面案例（扣分项）：说教式灌输、纯理论宣讲、强制接受观点
- 触发NEVER规则的说教式态度教学 → 扣维度D 5-10分

### 学情与马扎诺专项

**Q15: 马扎诺动机激发如何判定？**

A: 动机激发有效性判定：
- **有效动机激发**（加分/不扣分）：
  - 情境导入（职业情境、生活情境、问题情境）
  - 冲突认知（展示错误案例、抛出悖论）
  - 目标明确（学习目标与职业目标关联）
  - 成功预期（设置阶梯任务、可达成的微目标）
- **无效动机激发**（扣维度B 2-3分）：
  - 纯口头激励"同学们要认真听"
  - 无情境的强制集中注意力
  - 与教学内容无关的趣味元素堆砌

**Q16: 元认知训练和知识总结有什么区别？**

A: 关键区别在于是否引导学生"监控自己的学习"：
- **元认知训练**（符合要求）：
  - "刚才的解题思路你还记得吗？"
  - "这个方法还能用在哪些地方？"
  - "如果再做一遍，你会注意什么？"
  - 设计专门的反思环节（不少于3分钟）
- **普通知识总结**（不符合元认知）：
  - "今天学到了什么？"（仅回顾内容）
  - "把这节课的知识点再看一下"（仅复习）
  - 反思时间少于2分钟

### 教学评一致性专项

**Q17: 布鲁姆层级错位怎么判定？**

A: 目标与教学活动/评价的层级不一致：
- **目标定位高阶（分析/评价/创造）+ 活动仅低阶（记忆/理解）** → 扣维度H 3-5分
  - 例：目标"能够评价XX方案的优劣" → 活动仅"观看视频后回答问题"
- **目标低阶 + 活动高阶** → 扣维度H 2-3分
  - 例：目标"能说出XX的定义" → 活动"小组讨论并设计方案"
- **评价方式与目标不匹配** → 扣维度H 3-5分（见Q18）

**Q18: 目标与评价不一致的扣分怎么算？**

A: 一致性审计扣分规则：
- 目标-活动不匹配：每处扣2分，上限-6分
- 目标-评价不匹配：每处扣3分，上限-9分
- 三者完全不一致（目标/活动/评价层级全错）：额外扣2分
- **触发NEVER规则：无情境引入** → 扣维度B 3分；**反思仅为总结** → 扣维度B 2分

## 专业课专项问题

**Q21: 工作过程六属性必须全部出现吗？**

A: 不必须全部，但关键属性应出现：
- 至少含：资讯→实施→检查→评估
- 完整更好：资讯→计划→决策→实施→检查→评估

**Q22: 专业课思政融入有什么特殊要求？**

A: 需结合岗位职业道德：
- 护理专业：医德、患者关怀、安全操作
- 汽修专业：工匠精神、安全意识、质量观念
- 烹饪专业：食品安全、匠心传承、职业卫生
- 数控/机械：精度意识、安全操作、设备保养

## 报告生成问题

**Q23: 用户要求生成docx但无具体格式要求？**

A: 使用标准模板：
- 标题居中、字号36
- 表格带边框、列宽按规范
- 页脚含页码
- 八维度评分表需包含各维度得分与总分

**Q24: 评估后用户不想要报告文档？**

A: 仅显示对话中的摘要版本，不强制生成。

## NEVER规则汇总

**Q25: 哪些情况触发NEVER规则（硬伤扣分）？**

A: 以下5种情况为硬伤，直接触发维度D或H扣分：
1. **动作技能纯讲解**：讲授超过15分钟无示范无实操 → 扣维度D 10分
2. **智慧技能无练习**：仅讲规则/公式，无任何应用练习 → 扣维度D 5分
3. **态度说教式**：强制灌输态度观点，无体验无讨论 → 扣维度D 5分
4. **无情境引入**：直接开始讲授，无任何情境/问题/任务引入 → 扣维度B 3分
5. **反思仅为总结**：反思环节仅回顾知识点，无元认知深度反思 → 扣维度B 2分

## 边界情况

**Q26: 用户对评估结论不满意？**

A: 解释评分依据，指出具体扣分点，提供改进建议。

**Q27: 八维度总分如何计算？**

A: 八维度总分 = A基本信息(10) + B学情(12) + C教学目标(15) + D教法(10) + E活动(15) + F问题链(12) + G辅助(12) + H教学评一致性(14) = 100分
