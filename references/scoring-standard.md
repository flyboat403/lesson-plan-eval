# 评分标准与扣分细则

> 本文档定义教案评估的评分机制、扣分规则及具体示例。

---

## 评分机制（总分100分）

| 维度 | 分值 | 子项评分标准 |
|------|------|-------------|
| **基本信息审计** | 10分 | 课堂类型(3) + 课题规范(3) + 重难点措施(4) |
| **学情审计** | 12分 | 已有基础(2) + 预判困难(2) + 差异应对(2) + 动机激发(3) + 元认知训练(3) |
| **教学目标审计** | 15分 | 核心问题明确(4) + 可测性(4) + 岗课对接(4) + 思政融入(3) |
| **教法审计** | 10分 | 学习结果识别(3) + 教法匹配检查(7) |
| **活动审计** | 15分 | 方法选择(4) + 活动必要性(4) + 真假活动(3) + 考核明确(4) |
| **问题链审计** | 12分 | 无伪互动(4) + 思维递进(4) + 行动导向(4) |
| **辅助审计** | 12分 | 时间分配(4) + 板书设计(4) + 语言规范(2提醒) + 反思框架(2) |
| **教学评一致性审计** | 14分 | 目标内容匹配(4) + 目标评价匹配(4) + 布鲁姆层级(3) + 职业能力对接(3) |

## 扣分规则明细表

### 基础扣分规则（继承v0.3）

| 违规类型 | 扣分值 | 说明 |
|---------|-------|------|
| 目标散无核心问题 | -5分 | 核心硬伤，直接影响教学主线 |
| 使用模糊目标词 | -3分 | 每个模糊词扣1分，上限-3分 |
| 重难点无解决措施 | -3分 | 缺关键支撑设计 |
| 连续讲授超25分钟 | -4分 | 违反注意力曲线核心问题 |
| 教学方法单一 | -3分 | 不适合职教特点 |
| 形式活动无产出 | -4分 | 核心硬伤，浪费实训时间 |
| 活动不服务目标 | -3分 | 活动与目标脱节 |
| 考核标准不明确 | -2分 | 缺少评价依据 |
| 未标注工作属性【专业课】 | -3分 | 行动导向不完整 |
| 生硬贴思政标签 | -3分 | 思政融入不当 |
| 压缩评价时间 | -3分 | 反馈闭环缺失 |
| 伪互动提问 | -2分 | 每个无效提问扣1分，上限-2分 |
| 环节占比失衡 | -2分 | 时间分配问题 |
| 错别字/语病 | -1分 | 每处扣0.5分，上限-1分 |
| 目标无内容支撑 | -2分 | 每个无支撑目标扣2分，上限-5分 |
| 游离内容（教而无目标） | -1分 | 每处游离内容扣1分，上限-3分 |
| 目标无评价对应 | -2分 | 每个无评价目标扣2分，上限-5分 |
| 考而未教 | -2分 | 考核内容超出教学范围 |
| 布鲁姆层级错位 | -2分 | 目标层级与内容/评价层级不匹配 |
| 【专业课】目标内容评价分散 | -3分 | 三项未指向同一岗位能力 |

### 新增扣分规则（v0.4）

#### 维度D（教法审计）- 加涅匹配

| 违规类型 | 扣分值 | 说明 |
|---------|-------|------|
| **动作技能课程用纯讲解** | -4分 | 维度D核心硬伤，必须有示范—练习—反馈闭环 |
| **智慧技能课程无练习环节** | -3分 | 维度D教法不匹配，必须让学生动手应用 |
| **态度目标用说教式讲解** | -3分 | 维度D态度培养不当，必须有榜样、体验、强化 |
| **言语信息用角色扮演** | -2分 | 维度D教法过度设计，过度复杂化 |
| **学习结果类型识别错误** | -2分 | 维度D识别失败 |

#### 维度B（学情审计）- 马扎诺动机+元认知

| 违规类型 | 扣分值 | 说明 |
|---------|-------|------|
| **无情境引入** | -2分 | 维度B动机激发缺失，必须设计有价值的问题/情境 |
| **情境与专业脱节** | -1分 | 维度B情境设计问题 |
| **情境无吸引力/无思考任务** | -1分 | 维度B情境设计弱化 |
| **反思仅为知识总结** | -1分 | 维度B元认知训练不足，反思需有框架 |
| **无反思框架** | -2分 | 维度B元认知缺失 |

## 扣分计算示例

### 基础示例（继承v0.3）

| 场景描述 | 扣分计算 | 最终扣分 |
|---------|---------|---------|
| 目标中出现"了解""熟悉""掌握"三个模糊词 | 每词扣1分 → 1+1+1=3分 | **-3分** |
| 教师讲授28分钟（超出3分钟） | 超25分钟即扣满，不按分钟累加 | **-4分** |
| 存在"对不对""是不是"两个伪互动提问 | 每处扣1分 → 1+1=2分 | **-2分** |
| 目标散且无核心问题 + 两个模糊词 | -5（核心）+ -2（模糊词）= -7分 | **-7分** |
| 专业课未标注工作属性 + 压缩评价时间 | -3 + -3 = -6分 | **-6分** |
| 连续讲授30分钟 + 环节失衡（前松后紧） | -4 + -2 = -6分 | **-6分** |

### 新增示例（v0.4）

| 场景描述 | 扣分计算 | 最终扣分 |
|---------|---------|---------|
| 红酒斟倒（动作技能）课程全程讲解PPT，无示范无练习 | 动作技能纯讲解 → -4分 | **-4分** |
| 解方程（智慧技能）课程仅讲解公式，无练习环节 | 智慧技能无练习 → -3分 | **-3分** |
| 职业态度培养目标仅用"要树立正确态度"说教 | 态度目标说教式 → -3分 | **-3分** |
| 直接讲解概念，无情境引入 | 无情境引入 → -2分 | **-2分** |
| 课后反思仅为"本节课学习了XXX" | 反思仅为知识总结 → -1分 | **-1分** |
| 动作技能纯讲解 + 无情境引入 | -4 + -2 = -6分 | **-6分** |

## 总体结论分级标准

| 分值区间 | 结论 | 说明 | 行动建议 |
|---------|------|------|---------|
| **90-100分** | ⭐优秀 | 教案设计科学合理，可直接实施 | 无需修改，建议分享推广 |
| **75-89分** | ⭐良好 | 存在个别问题，稍作调整即可使用 | 按硬伤表逐条修正 |
| **60-74分** | ⭐合格 | 存在明显问题，需重点修改后使用 | 优先修正核心硬伤 |
| **<60分** | ⭐待改进 | 存在严重问题，需重新设计 | 建议重新设计或教研组研讨 |

---

## 评分计算公式

```
最终得分 = 基础分(100分) - 各项扣分总和

注：同一问题不重复扣分（如"目标散无核心问题"已扣-5，不额外扣"使用模糊词"）
注：扣分上限不超过该维度满分
```
