# 评分标准与扣分细则

> 本文档定义教案评估的评分机制、扣分规则及具体示例。

---

## 评分机制（总分100分）

| 维度 | 分值 | 子项评分标准 |
|------|------|-------------|
| **A 基本信息审计** | 10分 | 课堂类型(5) + 课题规范(5) |
| **B 学情审计** | 12分 | 已有基础(3) + 预判困难(3) + 差异应对(3) + 动机激发(3) + 元认知训练(3) |
| **C 教学目标审计** | 15分 | 核心问题(5) + 可测性(5) + 岗课对接(3) + 思政融入(2) |
| **D 教法审计** | 10分 | 学习结果识别(5) + 教法匹配(5) |
| **E 活动审计** | 15分 | 方法选择(4) + 活动必要性(4) + 真假活动(4) + 考核明确(3) |
| **F 问题链审计** | 12分 | 无伪互动(4) + 思维递进(4) + 行动导向(4) |
| **G 辅助审计** | 12分 | 时间分配(4) + 板书设计(3) + 语言规范(3) + 反思要求(2) |
| **H 教学评一致性审计** | 14分 | 目标内容匹配(5) + 目标评价匹配(5) + 布鲁姆层级一致(4) |

## 扣分规则明细表

### 基础扣分规则（继承v0.3）

| 违规类型 | 扣分值 | 说明 |
|---------|-------|------|
| **核心硬伤类** | | |
| 目标散无核心问题 | -5分 | 核心硬伤，直接影响教学主线 |
| 使用模糊目标词 | -3分 | 每个模糊词扣1分，上限-3分 |
| 重难点无解决措施 | -3分 | 缺关键支撑设计 |
| 形式活动无产出 | -4分 | 核心硬伤，浪费实训时间 |
| 活动不服务目标 | -3分 | 活动与目标脱节 |
| 考核标准不明确 | -2分 | 缺少评价依据 |
| **时间与注意力类** | | |
| 连续讲授超25分钟 | -4分 | 违反注意力曲线核心问题 |
| 压缩评价时间 | -3分 | 反馈闭环缺失 |
| 环节占比失衡 | -2分 | 时间分配问题 |
| **方法与教法类** | | |
| 教学方法单一 | -3分 | 不适合职教特点 |
| 动作技能纯讲解无实操 | -4分 | 加涅学习结果类型与教法不匹配 |
| 智慧技能纯讲解无练习 | -3分 | 智慧技能需实践操作 |

## 扣分计算示例

### 基础示例（继承v0.3）

| 场景描述 | 扣分计算 | 最终扣分 |
|---------|---------|---------|
| 目标中出现"了解""熟悉""掌握"三个模糊词 | 每词扣1分 → 1+1+1=3分 | **-3分** |
| 教师讲授28分钟（超出3分钟） | 超25分钟即扣满，不按分钟累加 | **-4分** |
| 存在"对不对""是不是"两个伪互动提问 | 每处扣1分 → 1+1=2分 | **-2分** |
| 动作技能课程（如"车削加工"）全程讲授无实操 | 动作技能纯讲解无实操 | **-4分** |
| 智慧技能课程（如"故障诊断"）纯讲授无练习 | 智慧技能纯讲解无练习 | **-3分** |
| 态度类课程（如"职业素养"）说教式教学 | 态度类纯说教无体验 | **-3分** |
| 直接进入知识讲解，无情境引入 | 无情境导入直接授课 | **-3分** |
| 反思环节仅写"本节课学会了..." | 反思仅为课堂总结 | **-2分** |
| 目标散且无核心问题 + 两个模糊词 | -5（核心）+ -2（模糊词）= -7分 | **-7分** |
| 专业课未标注工作属性 + 压缩评价时间 | -3 + -3 = -6分 | **-6分** |
| 连续讲授30分钟 + 环节失衡（前松后紧） | -4 + -2 = -6分 | **-6分** |

### 新增示例（v0.4）

| 场景描述 | 扣分计算 | 最终扣分 |
|---------|---------|---------|
| 红酒斟倒（动作技能）课程全程讲解PPT，无示范无练习 | 动作技能纯讲解 → -4分 | **-4分** |
| 解方程（智慧技能）课程仅讲解公式，无练习环节 | 智慧技能无练习 → -3分 | **-3分** |
| 职业态度培养目标仅用"要树立正确态度"说教 | 态度目标说教式 → -3分 | **-3分** |
| 直接讲解概念，无情境引入 | 无情境引入 → -2分 | **-2分** |
| 课后反思仅为"本节课学习了XXX" | 反思仅为知识总结 → -1分 | **-1分** |
| 动作技能纯讲解 + 无情境引入 | -4 + -2 = -6分 | **-6分** |

## 总体结论分级标准

| 分值区间 | 结论 | 说明 | 行动建议 |
|---------|------|------|---------|
| **90-100分** | ⭐优秀 | 教案设计科学合理，可直接实施 | 无需修改，建议分享推广 |
| **75-89分** | ⭐良好 | 存在个别问题，稍作调整即可使用 | 按硬伤表逐条修正 |
| **60-74分** | ⭐合格 | 存在明显问题，需重点修改后使用 | 优先修正核心硬伤 |
| **<60分** | ⭐待改进 | 存在严重问题，需重新设计 | 建议重新设计或教研组研讨 |

---

## 评分计算公式

```
最终得分 = 基础分(100分) - 各项扣分总和

注：同一问题不重复扣分（如"目标散无核心问题"已扣-5，不额外扣"使用模糊词"）
注：扣分上限不超过该维度满分（如A维度最多扣10分）
```
